== Description ==

This plugin are display current page title in browser tab title in wordpress admin section.



== Installation ==
Automatic installation through WordPress:

   1. Log-in to your WordPress site.
   2. Hover over Plugins and click Add New.
   3. Under Search type in wpadmin-page-browser-tab-title, then click Search Plugins.
   4. In the results page, click Install Now.
   5. Once installed, click Activate Plugin. You're done!
   
Manual installation:

   1. Upload the 'wpadmin-page-browser-tab-title' directory to the /wp-content/plugins/ directory.
   2. Go to your WordPress dashboard and navigate to Plugins -> Installed Plugins.
   3. On the Plugin page in your WordPress Administration area, activate the wpadmin-page-browser-tab-title plugin.
